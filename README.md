# Atividade_Projeto
Projeto Integrador: Análise de Dados com Python

# 1. Introdução
 Este projeto tem como objetivo aplicar conceitos de Python, estatística e storytelling na análise de dados de produtos. A base de dados contém informações sobre categorias, marcas, preços e descontos de diversos itens.

# 2. Referencial
- Técnicas aplicadas: análise estatística com média, mediana e desvio padrão, agrupamentos de dados e visualizações com gráficos interativos.
- Ferramentas: pandas, matplotlib/seaborn, plotly.
- Base teórica: conceitos vistos nas aulas do curso.
- Documentação consultada: pandas, seaborn, plotly, matplotlib.

# 3. Documentação do Projeto

## 3.1. Pré-requisitos
Ter as bibliotecas matplotlib, pandas, numpy, plotly.express instaladas.
